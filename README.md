# Калькулятор алюминиевых фасадов

Веб-приложение для менеджера по продаже алюминиевых профилей и расчёта стоимости мебельных фасадов со стеклом.

## Возможности

- Контактные данные клиента (имя, телефон, дата).
- Любое число фасадов в одном заказе.
- На каждый фасад: тип профиля (узкий/широкий), цвет профиля (4 варианта), цвет стекла (6 вариантов), размеры (В×Ш в мм), количество.
- Автоматический расчёт:
  - профиля по периметру (узкий 1900 ₸/м, широкий 2250 ₸/м);
  - стекла по площади (от 7 000 до 34 000 ₸/м²);
  - уголков (4 шт.), петель (по высоте), винтов, уплотнителя;
  - услуг: распил 45°, отверстия под петли, сборочные отверстия, сборка фасада;
  - на широком профиле петли не покупаются — только отверстия.
- Свод по позициям и итог по заказу.
- Печать / PDF из браузера.

## Запуск

Откройте `index.html` в любом современном браузере.

## Стек

Один HTML-файл — HTML + CSS + ванильный JavaScript. Без зависимостей, без сборки.
